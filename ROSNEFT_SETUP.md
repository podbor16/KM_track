### 4. Особенности трассы Роснефть

#### Тип маршрута: **Кольцевой (loop)**
- Бегуны движутся по кругу
- Для дистанции 5 км: 1 круг по трассе
- Для дистанции 10 км: 2 круга по трассе
- Для дистанции 3 км: 2 круга по укороченной трассе (1.5 км)

#### Расчет позиций
- Используется метод `get_position_on_loop()` вместо `get_shuttle_position()`
- Позиция рассчитывается как: `distance_on_current_lap = total_distance % route_length`

## Как использовать

### 1. Запуск сервера
```bash
cd server
python flask_server.py
```

### 2. Открытие карты Роснефть
Откройте в браузере:
```
http://localhost:5000/maps/rosneft.html
```

### 3. API Endpoints

#### Получение маршрута
```
GET /api/route?event=rosneft
```

Ответ:
```json
{
  "coordinates": [[56.0075399, 92.7246168], ...],
  "distance": 4.94,
  "way_id": 553966988,
  "event": "rosneft",
  "event_name": "Роснефть",
  "route_type": "loop"
}
```

#### Получение данных бегунов
```
GET /api/runners?event=rosneft
```

#### Список мероприятий
```
GET /api/events
```

## Обновление маршрута

Если нужно обновить маршрут из OpenStreetMap:

```bash
cd test_files
python get_rosneft_route.py
```

Это создаст/обновит файл `rosneft_route.json` с актуальными координатами.

## Добавление нового мероприятия

1. Найдите Way ID трассы в OpenStreetMap
2. Добавьте конфигурацию в `EVENTS_CONFIG` в `flask_server.py`
3. Создайте HTML файл в `maps/` (скопируйте `rosneft.html`)
4. Измените `CONFIG.EVENT_NAME` в HTML файле
5. При необходимости создайте JSON файл с маршрутом

## Troubleshooting

### Маршрут не отображается
- Проверьте наличие файла `rosneft_route.json` в корне проекта
- Проверьте консоль браузера на ошибки
- Убедитесь, что сервер запущен

### Маркеры не двигаются
- Проверьте, что в `race_data.json` есть данные участников
- Убедитесь, что параметр `event=rosneft` передается в API запросах

### Ошибка загрузки CSS
- Путь к CSS в `rosneft.html`: `../server/static/tracker.css`
- Убедитесь, что файл существует
